# ESP32 Web Server - LED Control

Simple project using an **ESP32 as a Web Server** to control an LED through a web page.

The ESP32 connects to a WiFi network and hosts a small webpage where you can turn the LED **ON** or **OFF** directly from your browser.

## 🚀 Features

- ESP32 Web Server
- LED control via browser
- Simple HTML + CSS interface
- WiFi connection
- Runs on port 80 (default HTTP port)

## 🧰 Technologies

- ESP32
- Arduino Framework
- HTML
- CSS
- WiFi Library (`WiFi.h`)

## 📷 Interface

The ESP32 serves a simple webpage with two buttons:

- 🟢 **LED ON**
- 🔴 **LED OFF**
